# Info Gizmo HUD

A customizable, clickable heads-up display (HUD) for Blender's 3D Viewport, designed to speed up your modeling workflow by putting essential tools right where you need them.

## Features
<img width="400" height="272" alt="image" src="https://github.com/user-attachments/assets/2297e066-afc9-4d01-aa35-4b75bb59bcba" />

- Quick Access
Toggle Object/Edit modes, Transform Orientations, Snap Elements, Snap Targets, and AutoMerge directly in the 3D Viewport.

- Theme Integration
The UI automatically matches your current Blender theme colors, specifically adapting to your Toggle UI settings.

- Smart Visibility
The Mode selector dynamically updates to show only the available modes based on your currently active object type (Mesh, Curve, Armature, etc.).

- Highly Customizable
Don't use certain snap elements? Hide them! You can toggle the visibility of individual buttons from the N-Panel to keep your workspace clean.

- Persistent Settings
Your HUD layout, visible elements, and position offsets are automatically saved to a JSON configuration file. They will be restored exactly as you left them, even across different blend files and sessions.

## Installation

1. Download the release zip file.
2. In Blender, go to Edit > Preferences > Add-ons.
3. Click Install... and select the downloaded zip file.
4. Check the box to enable the add-on named "3D View: --Info Gizmo: Snap / AutoMerge HUD--".

## How to Use
<img width="220" height="457" alt="image" src="https://github.com/user-attachments/assets/6751876c-ef62-4421-bf98-248acedbf0e7" />

1. Open the Sidebar (N-Panel) in the 3D Viewport by pressing N.
2. Navigate to the Gizmo HUD tab.
3. Click the Enable Clickable Toolbar button to activate the HUD.
4. Use the settings below the button to adjust the X and Y offsets, and choose which tools you want to display.
